Following are the step by step commands that you need to follow: 

*INSTALL APACHE*

sudo apt update
sudo apt install apache2

Adjust the Firewall to Allow Web Traffic:

sudo ufw app list
sudo ufw app info "Apache Full"

Create a gruop with your user and www-data user (helps with future permissions and ftp control)
sudo groupadd webgroup
sudo usermod -a -G webgroup myuser
sudo usermod -a -G webgroup www-data

Allow incoming HTTP and HTTPS traffic for this profile:
sudo ufw allow in "Apache Full"

*INSTALL MYSQL*

Here is the link to video that we created recently to install MYSQL:

sudo apt-get install mysql-server

go to your download folder
curl -OL https://dev.mysql.com/get/mysql-apt-config_0.8.12-1_all.deb
sudo dpkg -i mysql-apt-config*
sudo apt update

sudo mysql_secure_installation
choose a level of your password and falow the steps

*INSTALL PHP*

sudo apt install php7.3 libapache2-mod-php7.3 php7.3-mysql php7.3-zip php7.3-curl php7.3-mbstring
sudo gedit /etc/apache2/mods-enabled/dir.conf
sudo systemctl restart apache2
sudo systemctl status apache2

*TEST PHP*
sudo gedit /var/www/html/file.php

*INSTALL PHPMYADMIN*

sudo apt install phpmyadmin
sudo systemctl restart apache2

sudo cp /etc/phpmyadmin/apache.conf /etc/apache2/conf-available/phpmyadmin.conf

sudo a2enconf phpmyadmin
sudo systemctl restart apache2

If apt is not avaliable just force instalation from composer
sudo add-apt-repository ppa:phpmyadmin/ppa
sudo apt update
sudo apt install phpmyadmin
